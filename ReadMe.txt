				Python Password Generator using tkinter GUI
				
				
Simple password generator with GUI made in tkinter.

What does this program do? It generates a password on the basis of users choice of security (basic, medium, plus) and lenght.
BASIC will generate a password that consists of lowercase and uppercase letters.
MEDIUM will generate a password that consists of numbers and upper and lowercase letters.
PLUS will generate a passsword that consists of uppercase letters, lowercase letters, numbers and basic symbols.

Password lenght is limited to minimum of 8 and maximum of 16 character.
